# Installation

```
npm i gauravkabra
```


# Local Dev

### Check installation

```bash
$ node -v # v22.9.0
$ npm -v  # 10.8.3
```

### Dev

```
$ npm version [major, minor, patch]
```

### Run

```bash
$ node index.js
```

### Publish
Push changes to git. Then run:

```bash
$ npm login
$ npm publish
```
